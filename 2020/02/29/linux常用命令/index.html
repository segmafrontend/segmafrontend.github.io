vim

ctrl+b向上翻页

v进入视图模式，实现多选
ctrl+v进入列视图模式，实现多列选中

命令行

history查询输入命令历史
history number最近的多少条记录
!!执行上一条命令
!numer执行指定命令
ctrl+r搜索历史记录，搜索后继续按ctrl+r可翻页ß
ctrl+u清空当前输入
nohup command &nohup表示退出后仍然运行，&表示后台运行,文档
jobs -l查看当前终端后台进程
ps -aux查看进程,a所有程序,u用户为主,x不区分终端,文档
ctrl+z进程放到后台并暂停
bg <number>继续后台已经暂停的进程
fg <number>后台进程放到前台

  2. yum history undo <ID>卸载软件以及依赖
